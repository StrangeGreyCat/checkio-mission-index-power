Ti viene data una lista di numeri positivi ed un numero N. Devi trovare l'N-esima potenza dell'elemento della lista che ha indice N. Se l'indice N è fuori dalla lista allora restituisci -1. Non dimenticare che il primo elemento di una lista ha indice 0.

Vediamo alcuni esempi:
- array = [1, 2, 3, 4] ed N = 2, allora il risultato è 32 == 9;
- array = [1, 2, 3] ed N = 3, ma l'indice N è fuori dalla lista, dunque il risultato è -1.

Input: Due argomenti. Un elenco di numeri come una lista di interi ed un numero come un intero.

Output: Il risultato come un intero.

Esempio: 

index_power([1, 2, 3, 4], 2) == 9
index_power([1, 3, 10, 100], 3) == 1000000
index_power([0, 1], 0) == 1
index_power([1, 2], 3) == -1

Come si usa: Questa missione ti insegna l'uso elementare di liste ed indici combinandoli con della semplice matematica.

Prerequisiti: 0 < len(array) ≤ 10
0 ≤ N
all(0 ≤ x ≤ 100 for x in array)
