あなたには正の整数の配列とある数Nが与えられます。 あなたは配列の添え字Nの要素のN乗を見つけなければいけません。 Nが配列の外にあるときは-1を返してください。 最初の要素の添え字が0であることを忘れないようにしましょう。

例題を見てみましょう:
- 配列 = [1, 2, 3, 4] と N = 2, このとき結果は32 == 9です;
- 配列 = [1, 2, 3] と N = 3, しかしNは配列の外なので結果は-1になります。

入力: 2つのパラメータ、配列(整数のリスト)とN(整数)。

出力: 結果(整数)

例:

{% if interpreter.slug == "js-node" %} 
indexPower([1, 2, 3, 4], 2) == 9
indexPower([1, 3, 10, 100], 3) == 1000000
indexPower([0, 1], 0) == 1
indexPower([1, 2], 3) == -1
{% else %} 
index_power([1, 2, 3, 4], 2) == 9
index_power([1, 3, 10, 100], 3) == 1000000
index_power([0, 1], 0) == 1
index_power([1, 2], 3) == -1
{% endif %}

どのように使われるか: このミッションは、簡単な数学と組み合わせたときどのように基本的な配列と添え字を使うかについて教えてくれます。

事前条件: 0 < len(array) ≤ 10
0 ≤ N
all(0 ≤ x ≤ 100 for x in array)
